### Changes To Be Made

## Scoring

There are various issues with the current scoring system.

1. There is an excessive amount of ties between interactions. A tie should NEVER occur. A tiebreaking mechanism should be implemented as follows:

- Grok recieves the two tied interactions, and the following prompt:

'''
Here are two interactions between users. By your reasoning, which of the two would you describe as more nonsensical or otherwise "retarded"? Return JSON only: {"winner": 1} if the first is more retarded, {"winner": 2} if the second.

Interaction 1:
{interaction_1}

Interaction 2:
{interaction_2}

Which is more retarded?
'''

2. Currently, scores are all whole numbers. Modify the scoring algorithm to be more nuanced by using decimal numbers. Remember to prioritize interactions with low intellectual value.

## Chat messages

1. Currently, the bot jumps in to say 'Bottom x interaction of all time' quite often. Instead, rework this feature entirely. If it detects a intellectually devoid interaction or slur, have grok come up with a creative ridicule using the prompt in the reasoning section of this doc.

2. When the user uses /rate, it should now take an argument. that argument should represent the number of above messages to group together into one interaction. Consider what this means for the rest of the codebase.
